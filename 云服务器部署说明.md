# 云服务器部署说明

## 问题

部署到云服务器后，前端调用 `localhost:9000` 失败（ERR_CONNECTION_REFUSED）

## 原因

浏览器在用户本地运行，`localhost` 指向用户电脑，不是云服务器。

## 解决方案

使用 Nginx 反向代理：

```
浏览器
  ↓
Nginx :80
  ├─ /api/*     → Next.js :3000 (登录、注册)
  ├─ /py-api/*  → Python :9000 (AI 聊天)
  └─ /*         → Next.js :3000 (页面)
```

---

## 快速部署

### 1. 配置环境变量

```bash
cd web-app
nano .env.local
```

修改为：

```bash
# 云服务器部署
NEXT_PUBLIC_PYTHON_API_URL=/py-api
```

### 2. 运行部署脚本

```bash
cd ..
./deploy.sh
```

### 3. 访问系统

```
http://your-server-ip
```

---

## Nginx 配置解释

```nginx
# Next.js API (登录、注册)
location /api/ {
    proxy_pass http://localhost:3000;
}

# Python API (AI 聊天)
# /py-api/api/chat → localhost:9000/api/chat
location /py-api/ {
    rewrite ^/py-api/(.*) /$1 break;
    proxy_pass http://localhost:9000;
}

# 前端页面
location / {
    proxy_pass http://localhost:3000;
}
```

**请求示例**：
- 登录：`/api/auth/login` → Next.js :3000
- 聊天：`/py-api/api/chat` → Python :9000/api/chat
- 首页：`/` → Next.js :3000
