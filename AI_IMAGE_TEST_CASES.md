# AI 图像生成测试用例 - 股权结构图

## 📊 测试场景：VIE 架构股权穿透图

中概股公司（如阿里巴巴、腾讯）通常采用 **VIE（可变利益实体）架构**，通过开曼群岛、香港等多层架构实现境外上市。

---

### 提示词：
```
请生成一张阿里巴巴集团的股权结构图，简化版，自上而下包含四层：

第一层（顶部）：
- 境外投资者（软银、雅虎等）

第二层：
- Alibaba Group Holding Limited（开曼群岛注册）

第三层：
- 阿里巴巴香港有限公司

第四层（底部）：
- 浙江淘宝网络有限公司（国内运营实体）

用箭头连接各层级，箭头上标注"100%持股"或"协议控制"
风格要求：商业图表风格，蓝色和灰色配色，现代简约
```

### 预期效果：
- 清晰的四层架构
- 箭头指向明确
- 文字可读

## 🎓 VIE 架构知识补充

**为什么中概股需要 VIE 架构？**

1. **外资限制**：中国法律限制外资投资互联网、教育等行业
2. **境外上市需求**：公司希望在美国、香港上市融资
3. **解决方案**：
   - 境外设立控股公司（开曼群岛、香港）
   - 境内设立 WFOE（外商独资企业）
   - WFOE 通过协议控制境内运营实体（VIE）
   - 避免直接股权关系，通过合同实现控制

**典型 VIE 公司**：
- 阿里巴巴（NYSE: BABA）
- 腾讯（HKEX: 0700）
- 百度（NASDAQ: BIDU）
- 京东（NASDAQ: JD）
- 拼多多（NASDAQ: PDD）
